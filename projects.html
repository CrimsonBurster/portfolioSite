My First Game

The first game I worked on was simple, with the only objective being keep the aliens from destroying your ship for as long as possible. There isn't an actual way to beat it and it still has a few bugs, but generally works and was the foundation for learning how to code for myself. (Flash Required to Play)

A Requested Image

This is a banner image made for a guild in the mmorpg, Black Desert Online. After my guild leader posted a want ad for a new banner to bring in new recruits. I took it upon myself to create something with the retro synth-wave style. Using a mixture of inspiration from the game, guild members, and learning a lot of new photoshop techniques, this was the end result. The guild leader praised it and it is still being used today.

First Group Game

This is the first project I worked on with a group of people. This was a step forward in not only coding ability, but learning to work with other people. Having one member leave at the start and increasing the workload for the rest of the group. This project was helpful in learning to manage time, redistribute efforts, and learn to work with github as a collaboration. (Flash Required to Play)

Group Comic

A motion comic made as a group. The story follows someone caught in a nightmare, living through some of the lowest moments of their life again and how those obstacles were overcome. When coming up with the story, I wanted something that anyone could project themselves onto and connect with if they had been through hard times. Using the simple stick figures not only let everyone in the group contribute regardless of artistic ability, but allowed the main character to be a blank canvas for anyone going through it. (Flash Required to Play)
